# 产品需求文档

## 1. 产品定位

家庭内部财政审批小程序是一个夫妻双方使用的轻量共同资金审批工具。它不替代完整记账软件，而是解决“共同资金支出前需要说明、沟通、确认”的具体问题。

## 2. 目标用户

- 夫妻或长期共同生活伴侣。
- 家庭中一方主要负责资金管理，另一方偶尔需要申请共同资金。
- 双方希望减少口头沟通遗漏，让资金用途、金额和理由有记录可查。

## 3. 核心使用场景

### 场景 A：购买家庭用品

申请方想购买咖啡机、扫地机器人、儿童用品等，提交购买理由、金额、商品图片和链接。审批方查看后决定是否通过。

### 场景 B：临时较大支出

申请方需要一笔临时资金，例如维修、学习课程、出行费用。审批方根据说明与金额进行判断。

### 场景 C：复盘共同支出

双方月底查看已通过和已驳回申请，了解共同资金的主要去向。

## 4. MVP 功能范围

### 4.1 申请方功能

- 创建资金申请。
- 填写资金用途。
- 填写申请金额。
- 填写购买必要性说明。
- 上传一张或多张产品照片。
- 粘贴产品链接。
- 查看申请当前状态。
- 查看审批意见。

### 4.2 审批方功能

- 查看待审批申请列表。
- 查看申请详情。
- 通过申请。
- 驳回申请。
- 填写审批意见。

### 4.3 双方通用功能

- 查看申请历史。
- 按状态筛选申请。
- 查看申请创建时间、审批时间。
- 首次进入时选择性别主题：男生淡蓝、女生淡粉。
- 选择自己在小程序中的角色：申请者或审批者。

## 5. 非 MVP 功能

以下功能暂不进入第一版，但保留扩展空间：

- 微信订阅消息提醒。
- 月度预算规则。
- 分类预算上限。
- 多家庭成员。
- 付款凭证上传。
- 自动抓取商品链接信息。
- AI 辅助整理购买理由。
- 账本统计图表。

## 6. 申请字段

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| 用途标题 | 是 | 例如“购买咖啡机” |
| 申请金额 | 是 | 单位为元 |
| 支出分类 | 否 | 家电、餐饮、学习、旅行、维修等 |
| 必要性说明 | 是 | 为什么需要买、现在买是否必要 |
| 产品照片 | 否 | 商品图、截图或现场照片 |
| 产品链接 | 否 | 电商链接或参考页面 |
| 期望购买时间 | 否 | 希望何时使用资金 |
| 备注 | 否 | 其他补充信息 |

## 7. 申请状态

- 草稿：申请方尚未提交。
- 待审批：申请已提交，等待审批方处理。
- 已通过：审批方同意使用共同资金。
- 已驳回：审批方不同意，通常附带原因。
- 已撤回：申请方主动撤回。

MVP 可以先实现：待审批、已通过、已驳回。

## 8. 关键流程

### 8.0 首次设置

1. 用户进入“共同花”。
2. 选择性别主题：男生淡蓝或女生淡粉。
3. 进入对应主题色的角色选择页。
4. 选择申请者或审批者。
5. 进入主界面，后续页面使用对应主题色。

### 8.1 提交申请

1. 申请方进入“新建申请”。
2. 填写用途、金额、必要性说明。
3. 可选上传照片和填写链接。
4. 点击提交。
5. 系统生成待审批申请。

### 8.2 审批申请

1. 审批方进入首页或申请列表。
2. 打开待审批申请详情。
3. 阅读金额、用途、必要性、照片、链接。
4. 填写审批意见。
5. 点击通过或驳回。
6. 系统更新状态并记录审批时间。

## 9. 验收标准

- 可以创建一条包含用途、金额、说明、图片、链接的申请。
- 创建后能在列表中看到该申请。
- 申请详情页展示完整申请信息。
- 待审批申请可以被通过或驳回。
- 状态变化后列表与详情页同步更新。
- 本地模拟数据能支撑完整演示流程。
