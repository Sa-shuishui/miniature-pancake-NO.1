# 视觉方向备选

当前还没有指定 Figma、截图、品牌或竞品参考。根据产品定位，建议先在下面三种方向中选择一种，再进入小程序 UI 实现。

## 方向 A：温馨家庭感

适合强调“夫妻沟通”和“共同决策”。

- 色彩：暖白、浅杏、柔和绿色作为点缀。
- 气质：轻松、亲密、没有压迫感。
- 页面感受：像家庭共享清单，而不是严肃财务系统。
- 适合文案：更口语化，例如“这笔申请想和你商量一下”。
- 风险：如果太温馨，金额审批的严肃性会弱一些。

## 方向 B：专业财务感

适合强调“共同资金要有规则”。

- 色彩：白底、深灰文字、蓝绿色强调色。
- 气质：清楚、可靠、像轻量审批工作台。
- 页面感受：信息密度更高，金额、状态、审批动作突出。
- 适合文案：更直接，例如“提交资金申请”“审批通过”。
- 风险：可能略显办公化，家庭情绪温度少一些。

## 方向 C：极简微信原生感

适合快速上线、降低学习成本。

- 色彩：接近微信生态常见的白、浅灰、绿色。
- 气质：熟悉、简洁、操作路径短。
- 页面感受：表单和列表都保持小程序原生习惯。
- 适合文案：短句为主，例如“申请”“审批”“历史”。
- 风险：辨识度较弱，但最容易开发和维护。

## 建议选择

如果目标是尽快做出可用 MVP，建议选择方向 C。

如果你希望这个工具更像一个“夫妻共同协商空间”，建议选择方向 A。

如果你希望它更像严肃的家庭财务制度，建议选择方向 B。
