# 上线检查清单

## 1. 小程序基础信息

- 小程序名称已确定。
- 小程序头像已准备。
- 服务类目选择合理。
- 简介说明清楚，不夸大功能。
- 客服联系方式已配置。

## 2. 隐私与数据

- 隐私政策说明收集的信息。
- 明确说明会保存申请用途、金额、说明、图片、链接和审批意见。
- 不收集无关通讯录、定位等敏感信息。
- 图片只用于申请说明。
- 删除家庭或退出家庭的数据策略已明确。

## 3. 权限与安全

- 云数据库集合权限已限制。
- 云函数会校验调用者身份。
- 家庭成员只能访问本家庭数据。
- 审批操作会校验当前用户是否有审批权限。
- 客户端不保存敏感密钥。

## 4. 审批流程

- 待审批申请能正常展示。
- 通过申请会记录审批人、审批意见、审批时间。
- 驳回申请会记录审批人、驳回原因、审批时间。
- 已审批申请不会重复审批。
- 申请状态在首页、列表、详情页一致。

## 5. 表单与内容

- 用途标题不能为空。
- 金额必须大于 0。
- 必要性说明不能为空。
- 产品链接格式正确。
- 图片数量限制明确。
- 图片上传失败时有提示。

## 6. 兼容性

- iPhone 小屏真机测试通过。
- Android 常见机型测试通过。
- 微信开发者工具基础库版本已确认。
- 底部安全区和键盘弹起不遮挡关键按钮。

## 7. 发布前回归

- 完整跑通创建申请。
- 完整跑通审批通过。
- 完整跑通审批驳回。
- 完整跑通列表筛选。
- 断网或云函数失败时能提示。
- 审核版本无调试日志和测试账号敏感信息。
